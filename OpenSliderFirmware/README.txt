OPEN SLIDER FIRMWARE
Written by Greg Deeth

This firmware was designed for the Arduino Mega 2560 + RAMPS 1.4 Shield

Requires the following libraries to be loaded into the Arduino database:
Timer One		Use the library manager
Timer Three		Use the library manager

SKETCH > INCLUDE LIBRARY > MANAGE LIBRARIES	Search for the two libraries above then install them.

Please compile this sketch and upload it to your device with the Arduino IDE programming software.

If you do not have the Arduino IDE, you can find it here:
https://www.arduino.cc/en/Main/Software


If you need help, please reach out to the community for answers:
https://sourceforge.net/p/open-slider/discussion/
